# Facebook Live Sellers Dataset Analysis

## Description
This project analyzes the Facebook Live Sellers Dataset to understand user engagement based on reactions, comments, and shares. It also applies the K-Means clustering algorithm to group posts with similar engagement patterns.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook / Google Colab

## Dataset
Facebook Live Sellers Dataset
